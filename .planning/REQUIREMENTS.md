# Requirements: GMStudio — Site Portfólio

**Defined:** 2026-03-30
**Core Value:** Um visitante que chega ao site precisa sair convencido de que a GMStudio é a escolha certa para seu projeto digital — e saber exatamente como entrar em contato.

## v1 Requirements

### Identidade Visual (BRAND)

- [ ] **BRAND-01**: Logo criado com variações (principal, monocromático, favicon)
- [ ] **BRAND-02**: Paleta de cores definida como tokens CSS (primárias, secundárias, neutras)
- [ ] **BRAND-03**: Tipografia definida (fontes para títulos e corpo de texto)
- [x] **BRAND-04**: Guia de estilo básico documentando uso correto de logo, cores e fontes

### Hero & Base (HERO)

- [ ] **HERO-01**: Seção hero com proposta de valor clara e CTA de contato visível
- [ ] **HERO-02**: Site responsivo com layout mobile-first

### Portfólio (PORTF)

- [ ] **PORTF-01**: Grid de projetos com thumbnail, nome e categoria de cada case
- [ ] **PORTF-02**: Cada projeto com link externo para visitar o site ao vivo

### Serviços (SERV)

- [ ] **SERV-01**: Cards de serviços com ícone e descrição para cada tipo de site entregue (landing page, loja, cardápio, institucional)
- [ ] **SERV-02**: Seção de processo de trabalho mostrando as etapas (briefing → desenvolvimento → entrega)

### Sobre & Equipe (ABOUT)

- [ ] **ABOUT-01**: Seção com história da agência (quem somos, quando começamos, missão)
- [ ] **ABOUT-02**: Cards da equipe com foto, nome e função de cada membro
- [ ] **ABOUT-03**: Números e conquistas (projetos entregues, clientes atendidos, anos de experiência)
- [ ] **ABOUT-04**: Valores da agência apresentados na seção sobre

### Depoimentos (TESTI)

- [ ] **TESTI-01**: Carrossel/slider de depoimentos com foto e nome do cliente

### Contato (CONT)

- [ ] **CONT-01**: Botão WhatsApp flutuante fixo na tela com mensagem pré-preenchida
- [ ] **CONT-02**: Formulário de contato com campos: nome, email, tipo de projeto e mensagem
- [ ] **CONT-03**: Links para redes sociais (Instagram, LinkedIn) na seção de contato e/ou rodapé
- [ ] **CONT-04**: Email da agência exibido diretamente na seção de contato

### SEO & Performance (SEO)

- [ ] **SEO-01**: Open Graph tags configuradas para preview correto ao compartilhar no WhatsApp e redes sociais
- [ ] **SEO-02**: Imagens otimizadas (formato WebP, dimensões corretas, lazy loading)
- [ ] **SEO-03**: Sitemap.xml e robots.txt configurados para indexação no Google

## v2 Requirements

### Portfólio Avançado

- **PORTF-03**: Filtro de projetos por categoria (loja, cardápio, landing page, institucional)
- **PORTF-04**: Cada case com contexto de negócio (problema do cliente + resultado obtido)
- **PORTF-05**: Páginas individuais de case study com detalhamento completo

### Serviços Avançados

- **SERV-03**: Lista detalhada do que está incluído em cada serviço (domínio, hospedagem, manutenção, etc.)
- **SERV-04**: Sinal de preço ("a partir de R$...") por tipo de projeto

### SEO Avançado

- **SEO-04**: Meta tags SEO completas (título otimizado, meta descrição, robots por página)

### Social Proof

- **TESTI-02**: Avaliação por estrelas junto a cada depoimento

## Out of Scope

| Feature | Reason |
|---------|--------|
| Blog / conteúdo editorial | Desvio do foco de captação — complexidade sem retorno direto no v1 |
| Área do cliente / autenticação | Desnecessário para portfólio/vitrine |
| Painel CMS para edição de conteúdo | Site estático por ora — CMS pode vir em milestone futura |
| E-commerce / pagamento no portfólio | O portfólio não é a loja — os projetos do cliente são |
| Chatbot | Adiciona complexidade e pode piorar experiência mobile |
| Animações pesadas | Prejudica performance no mobile e LCP — micro-animações apenas se não impactarem Core Web Vitals |
