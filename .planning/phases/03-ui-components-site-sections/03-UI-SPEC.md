## Spacing Scale

All values from `tokens.css`. Multiples of 4.

| Token | CSS Var | Value | Usage in Phase 3 |
|-------|---------|-------|-----------------|
| xs | `--space-1` | 4px | Icon gaps, badge padding, inline label gaps |
| sm | `--space-2` | 8px | Compact element spacing, button icon gaps |
| md | `--space-4` | 16px | Default element padding, card inner padding |
| lg | `--space-6` | 24px | Grid gaps between cards, nav item spacing |
| xl | `--space-8` | 32px | Section inner padding top/bottom (mobile) |
| 2xl | `--space-12` | 48px | Section padding top/bottom (desktop) |
| 3xl | `--space-16` | 64px | Section vertical gap between sections |
| 4xl | `--space-20` | 80px | Hero section top padding |
| 5xl | `--space-24` | 96px | Hero section bottom padding |

**Exceptions:**

- Touch targets (nav hamburger, floating WhatsApp button, CTA buttons): minimum 44px height. Use `min-h-[44px]` Tailwind utility.
- Section-to-section vertical rhythm: `--space-24` (96px) on desktop, `--space-16` (64px) on mobile.
- Nav height: 64px fixed (`--space-16`).

## Typography

All roles use tokens from `tokens.css`. Heading roles use `font-heading` (Space Grotesk). Body roles use `font-body` (Inter).

| Role | Size | CSS Var | Weight | Font | Line Height | Tracking | Usage |
|------|------|---------|--------|------|-------------|----------|-------|
| Display | 60px (desktop) / 48px (mobile) | `--text-6xl` / `--text-5xl` | 700 | Space Grotesk | 1.1 (`--leading-tight`) | -0.025em (`--tracking-tight`) | Hero headline |
| Heading | 36px (desktop) / 30px (mobile) | `--text-4xl` / `--text-3xl` | 700 | Space Grotesk | 1.2 (`--leading-snug`) | -0.025em | Section headings (Services, Portfolio, About, Contact) |
| Subheading | 24px | `--text-2xl` | 600 | Space Grotesk | 1.2 (`--leading-snug`) | 0 | Card titles, process step headings, team member names |
| Card Title | 20px | `--text-xl` | 600 | Space Grotesk | 1.2 | 0 | Service card title, portfolio project name |
| Body | 16px | `--text-base` | 400 | Inter | 1.5 (`--leading-normal`) | 0 | Paragraphs, descriptions, about text, testimonials |
| Small | 14px | `--text-sm` | 400 | Inter | 1.5 | 0 | Captions, category tags, meta text, muted labels |
| Label | 12px | `--text-xs` | 500 | Inter | 1.5 | 0.05em (`--tracking-wide`) | Uppercase badge text, nav link labels on mobile |

**Weight constraint:** Only 400 (regular) and 700 (bold) for body/display contexts. 600 (semibold) permitted only for card titles and subheadings. No other weights in use.

**Hero subtitle:** 18px (`--text-lg`), weight 400, Inter, line-height 1.5. Color: `--color-text-muted`.

## Color

All values from `tokens.css`. Dark premium palette (D-01, D-02).

| Role | CSS Var | Hex | Usage |
|------|---------|-----|-------|
| Dominant (60%) | `--color-bg` | `#0A0A0F` | Page background, nav background (opaque), hero background |
| Secondary (30%) | `--color-surface` | `#13131A` | Card surfaces, glassmorphism base, contact section background |
| Surface raised | `--color-surface-raised` | `#1A1A24` | Modal, tooltip, active nav drawer background |
| Accent (10%) | `--color-accent` | `#6366F1` | See reserved list below |
| Accent hover | `--color-accent-hover` | `#4F46E5` | Accent elements on hover/focus |
| Accent surface | `--color-accent-surface` | `#1E1B4B` | Category tag background, badge tint |
| Text primary | `--color-text` | `#F8F8F8` | All primary text |
| Text muted | `--color-text-muted` | `#9CA3AF` | Subtitles, captions, meta, hero subtitle |
| Text subtle | `--color-text-subtle` | `#6B7280` | Placeholders, disabled states, form hints |
| Border | `--color-border` | `rgba(248,248,248,0.08)` | All card borders, glassmorphism hairline, nav bottom border |
| Border accent | `--color-border-accent` | `rgba(99,102,241,0.30)` | Card border on hover, CTA button outline on hover |
| Destructive | — | `#EF4444` (Tailwind red-500) | Form error state text and border only |

**Accent (`#6366F1`) reserved for — and only for:**

1. Primary CTA button background (WhatsApp button in hero, primary form submit)
2. Floating WhatsApp button background
3. Indigo radial glow behind hero headline (`background: radial-gradient`)
4. Nav active link underline indicator
5. Accent glow on primary CTA hover (`--shadow-glow`: `0 0 24px rgba(99,102,241,0.25)`)
6. Logo "GM" letters
7. Category tag text color (on `--color-accent-surface` background)
8. Section heading accent word or underline decoration (sparingly, one word per heading if used)

**Accent is NOT used for:** body links, icon fills (except logo), all other borders, background fills outside the above list.

## Glassmorphism Contract

All cards in this phase use the glassmorphism pattern (D-05, D-STYLE-02).

| Property | Value | CSS Var |
|----------|-------|---------|
| Background | `rgba(19, 19, 26, 0.60)` | `--glass-bg` |
| Border | `rgba(248, 248, 248, 0.08)` | `--glass-border` |
| Backdrop filter | `blur(12px)` | `--glass-blur` |
| Border radius | 12px | `--radius-lg` |
| Shadow | `0 4px 16px rgba(0,0,0,0.4)` | `--shadow-md` |

**Card hover state:**

- Border upgrades to `--color-border-accent` (`rgba(99,102,241,0.30)`)
- Box shadow upgrades to `--shadow-lg` (`0 8px 32px rgba(0,0,0,0.5)`) + `--shadow-glow`
- Transform: `translateY(-2px)`
- Transition: `--transition-base` (200ms ease)

No JS for hover — Tailwind `transition` utilities only (D-ANIM-03).

## Animation Contract

All animations: zero JS, no Motion library. CSS only (D-ANIM-02).

### Section Entrance (Fade-Up)

```css
@keyframes fade-up {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}
```

| Property | Value |
|----------|-------|
| Trigger | Intersection Observer API — inline `<script>` tag in each section component |
| Default state | `opacity: 0; transform: translateY(20px)` |
| Active state | Class `.visible` added → `animation: fade-up 0.5s ease forwards` |
| Duration | 500ms |
| Easing | `ease` |
| Stagger (multiple children) | `animation-delay: calc(var(--i, 0) * 80ms)` via inline CSS var |
| Threshold | 0.15 (15% of element visible) |

### Card Hover

| Property | Value |
|----------|-------|
| Transform | `translateY(-2px)` |
| Border | Upgrades to `--color-border-accent` |
| Shadow | `--shadow-lg` + `--shadow-glow` |
| Transition | `transition: all 200ms ease` (Tailwind `transition` utility) |
| Implementation | Tailwind `group-hover` or direct `hover:` utilities — no JS |

### Testimonial Marquee

| Property | Value |
|----------|-------|
| Duration | 30s linear infinite |
| Pause on hover | `animation-play-state: paused` on track hover — CSS only |
| Reduced motion | `@media (prefers-reduced-motion: reduce) { animation: none; overflow-x: auto; }` |

### Hero Glow

| Property | Value |
|----------|-------|
| Type | Static CSS gradient — no animation |
| Value | `radial-gradient(ellipse 80% 50% at 50% -10%, rgba(99,102,241,0.25), transparent)` |

## Accessibility Contract

| Rule | Value |
|------|-------|
| Color contrast body text | `--color-text` (#F8F8F8) on `--color-bg` (#0A0A0F) — ratio ~19:1. WCAG AAA. |
| Color contrast muted text | `--color-text-muted` (#9CA3AF) on `--color-bg` (#0A0A0F) — ratio ~5.9:1. WCAG AA. |
| Color contrast accent text | `--color-accent` (#6366F1) on `--color-bg` — ratio ~3.5:1. Use accent for decorative/large text only, not body. |
| Focus ring | `focus-visible:outline` — 2px `--color-accent` offset 2px. Never suppress focus outlines. |
| Touch targets | Minimum 44px height/width for all interactive elements |
| Reduced motion | `@media (prefers-reduced-motion: reduce)` disables marquee, fade-up animations |
| Nav mobile | `aria-expanded` on hamburger button. Drawer: `role="dialog"` or `role="navigation"`. |
| Form | All inputs have `<label>` elements (visible or sr-only). `aria-required="true"` on required fields. `aria-describedby` linking to error messages. |
| Images | All `<Image />` components require meaningful `alt` text. Decorative images: `alt=""`. |
| External links | `rel="noopener noreferrer"` on all `target="_blank"` links. Screen reader note: add `aria-label="Visitar {project name} (abre em nova aba)"`. |
| Landmark roles | `<header>`, `<main>`, `<footer>`, `<nav>`, `<section aria-labelledby>` on each section. |
