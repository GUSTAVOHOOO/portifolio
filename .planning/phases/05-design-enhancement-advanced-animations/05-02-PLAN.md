# Plan 05-02: Hero Visual Upgrade

**Phase:** 5 — Design Enhancement & Advanced Animations
**Goal:** Replace the static hero radial-gradient with a CSS animated mesh gradient background and add Motion-powered staggered word reveal on the headline and spring entrance on the CTAs, using a React island for the animated content.
**Depends on:** Plan 05-01 (motion installed, old animation system removed)

---

## Success Criteria

- [ ] The hero background has a visible, smooth animated gradient effect (no canvas, no WebGL, no Three.js)
- [ ] The headline "Presença digital que converte" reveals word by word with a staggered spring animation on page load
- [ ] The two CTA buttons (`Falar no WhatsApp` and `Ver portfólio`) entrance with a spring bounce (scale 0.8 → 1, opacity 0 → 1) after the headline animation completes
- [ ] `whileTap` scale-down feedback (scale: 0.95) works on the primary CTA
- [ ] Without JavaScript, the hero section is fully readable (static fallback: headline and CTAs are visible via a `<noscript>` or SSR-visible wrapper)
- [ ] `npm run build` produces zero errors
- [ ] No Three.js, `@react-three/fiber`, or `three` imports in any new file

## Tasks

### Task 1: Add CSS animated mesh gradient background to HeroSection.astro

**File:** `src/components/HeroSection.astro`

The current hero background is set via inline `style` on the `<section>` element:

```css
background:
  radial-gradient(ellipse 80% 50% at 50% -10%, rgba(99,102,241,0.25), transparent),
  var(--color-bg);
```

Replace the hero `<section>` background approach as follows:

1. Remove the `background` property from the inline `style` attribute on `<section id="hero"`.

2. Add `class="hero-bg-animated"` to the `<section id="hero"` element so the section becomes:
   ```html
   <section
     id="hero"
     class="hero-bg-animated"
     style="
       min-height: 100vh;
       display: flex;
       align-items: center;
       justify-content: center;
       text-align: center;
       padding-top: 80px;
       padding-bottom: 96px;
       position: relative;
     "
   >
   ```

3. Inside the `<style>` block of `HeroSection.astro` (create one if it doesn't already have the right rule), add the animated gradient:

```css
@keyframes hero-mesh-shift {
  0%   { background-position: 0% 50%; }
  50%  { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

.hero-bg-animated {
  background:
    radial-gradient(ellipse 120% 80% at 60% -20%, rgba(99, 102, 241, 0.30), transparent 60%),
    radial-gradient(ellipse 80% 60% at 10% 80%, rgba(139, 92, 246, 0.20), transparent 55%),
    radial-gradient(ellipse 60% 40% at 90% 60%, rgba(99, 102, 241, 0.15), transparent 50%),
    var(--color-bg);
  background-size: 300% 300%;
  animation: hero-mesh-shift 12s ease infinite;
}
```

Note: The `.hero-bg-animated` reduced-motion disable is already in `global.css` from Plan 05-01. No need to re-add it here.

### Task 2: Create HeroContent React island with Motion animations

**File:** `src/components/HeroContent.tsx` (new file)

Create this new React component. It handles the headline word-reveal and CTA spring entrance using `motion` v12:

```tsx
import { useReducedMotion, LazyMotion, domAnimation, AnimatePresence } from "motion/react"
import * as m from "motion/react-m"

const whatsappPhone = '5543996142514'
const whatsappMessage = encodeURIComponent('Olá, vim pelo site da GMStudio')
const whatsappUrl = `https://wa.me/${whatsappPhone}?text=${whatsappMessage}`

const headlineWords = ['Presença', 'digital', 'que', 'converte']

const containerVariants = {
  hidden: {},
  show: {
    transition: {
      staggerChildren: 0.12,
      delayChildren: 0.1,
    },
  },
}

const wordVariants = {
  hidden: { opacity: 0, y: 28, filter: 'blur(4px)' },
  show: {
    opacity: 1,
    y: 0,
    filter: 'blur(0px)',
    transition: { type: 'spring' as const, stiffness: 300, damping: 28 },
  },
}

const ctaVariants = {
  hidden: { opacity: 0, scale: 0.85 },
  show: {
    opacity: 1,
    scale: 1,
    transition: { type: 'spring' as const, stiffness: 400, damping: 20 },
  },
}

export default function HeroContent() {
  const prefersReduced = useReducedMotion()

  return (
    <LazyMotion features={domAnimation}>
      {/* Headline with word-by-word reveal */}
      <m.h1
        className="hero-heading"
        variants={prefersReduced ? undefined : containerVariants}
        initial={prefersReduced ? false : 'hidden'}
        animate="show"
        aria-label="Presença digital que converte"
        style={{
          fontFamily: 'var(--font-heading)',
          fontSize: 'var(--text-5xl)',
          fontWeight: 700,
          lineHeight: 'var(--leading-tight)',
          letterSpacing: 'var(--tracking-tight)',
          color: 'var(--color-text)',
          margin: 0,
          display: 'flex',
          flexWrap: 'wrap',
          justifyContent: 'center',
          gap: '0 0.28em',
        }}
      >
        {headlineWords.map((word) => (
          <m.span
            key={word}
            variants={prefersReduced ? undefined : wordVariants}
            style={{ display: 'inline-block' }}
          >
            {word}
          </m.span>
        ))}
      </m.h1>

      {/* Subtitle */}
      <m.p
        initial={prefersReduced ? false : { opacity: 0, y: 16 }}
        animate={{ opacity: 1, y: 0 }}
        transition={{ delay: prefersReduced ? 0 : 0.6, duration: 0.5, ease: 'easeOut' }}
        style={{
          fontFamily: 'var(--font-body)',
          fontSize: 'var(--text-base)',
          fontWeight: 400,
          color: 'var(--color-text-muted)',
          lineHeight: 'var(--leading-normal)',
          maxWidth: '560px',
          margin: 'var(--space-6) auto 0',
        }}
      >
        Criamos sites e lojas online que geram resultados reais para o seu negócio.
      </m.p>

      {/* CTA buttons */}
      <m.div
        variants={prefersReduced ? undefined : {
          hidden: {},
          show: { transition: { staggerChildren: 0.1, delayChildren: 0.72 } },
        }}
        initial={prefersReduced ? false : 'hidden'}
        animate="show"
        style={{
          display: 'flex',
          flexWrap: 'wrap',
          gap: 'var(--space-4)',
          justifyContent: 'center',
          marginTop: 'var(--space-8)',
        }}
      >
        {/* Primary CTA: WhatsApp */}
        <m.a
          variants={prefersReduced ? undefined : ctaVariants}
          whileTap={prefersReduced ? undefined : { scale: 0.95 }}
          href={whatsappUrl}
          target="_blank"
          rel="noopener noreferrer"
          style={{
            background: 'var(--color-accent)',
            borderRadius: 'var(--radius-full)',
            minHeight: '44px',
            padding: '0 var(--space-8)',
            fontFamily: 'var(--font-heading)',
            fontWeight: 700,
            fontSize: 'var(--text-base)',
            color: 'var(--color-text)',
            textDecoration: 'none',
            display: 'inline-flex',
            alignItems: 'center',
            gap: 'var(--space-2)',
            transition: 'background var(--transition-base), box-shadow var(--transition-base)',
          }}
          onMouseEnter={(e) => {
            const el = e.currentTarget as HTMLAnchorElement
            el.style.background = 'var(--color-accent-hover)'
            el.style.boxShadow = 'var(--shadow-glow)'
          }}
          onMouseLeave={(e) => {
            const el = e.currentTarget as HTMLAnchorElement
            el.style.background = 'var(--color-accent)'
            el.style.boxShadow = 'none'
          }}
        >
          <svg
            xmlns="http://www.w3.org/2000/svg"
            width="18"
            height="18"
            viewBox="0 0 24 24"
            fill="white"
            aria-hidden="true"
          >
            <path d="M17.472 14.382c-.297-.149-1.758-.867-2.03-.967-.273-.099-.471-.148-.67.15-.197.297-.767.966-.94 1.164-.173.199-.347.223-.644.075-.297-.15-1.255-.463-2.39-1.475-.883-.788-1.48-1.761-1.653-2.059-.173-.297-.018-.458.13-.606.134-.133.298-.347.446-.52.149-.174.198-.298.298-.497.099-.198.05-.371-.025-.52-.075-.149-.669-1.612-.916-2.207-.242-.579-.487-.5-.669-.51-.173-.008-.371-.01-.57-.01-.198 0-.52.074-.792.372-.272.297-1.04 1.016-1.04 2.479 0 1.462 1.065 2.875 1.213 3.074.149.198 2.096 3.2 5.077 4.487.709.306 1.262.489 1.694.625.712.227 1.36.195 1.871.118.571-.085 1.758-.719 2.006-1.413.248-.694.248-1.289.173-1.413-.074-.124-.272-.198-.57-.347m-5.421 7.403h-.004a9.87 9.87 0 01-5.031-1.378l-.361-.214-3.741.982.998-3.648-.235-.374a9.86 9.86 0 01-1.51-5.26c.001-5.45 4.436-9.884 9.888-9.884 2.64 0 5.122 1.03 6.988 2.898a9.825 9.825 0 012.893 6.994c-.003 5.45-4.437 9.884-9.885 9.884m8.413-18.297A11.815 11.815 0 0012.05 0C5.495 0 .16 5.335.157 11.892c0 2.096.547 4.142 1.588 5.945L.057 24l6.305-1.654a11.882 11.882 0 005.683 1.448h.005c6.554 0 11.89-5.335 11.893-11.893a11.821 11.821 0 00-3.48-8.413z"/>
          </svg>
          Falar no WhatsApp
        </m.a>

        {/* Secondary CTA: Ver portfólio */}
        <m.a
          variants={prefersReduced ? undefined : ctaVariants}
          href="#portfolio"
          style={{
            border: '1px solid var(--color-border)',
            borderRadius: 'var(--radius-full)',
            minHeight: '44px',
            padding: '0 var(--space-8)',
            fontFamily: 'var(--font-heading)',
            fontWeight: 700,
            fontSize: 'var(--text-base)',
            background: 'transparent',
            color: 'var(--color-text)',
            textDecoration: 'none',
            display: 'inline-flex',
            alignItems: 'center',
            transition: 'border-color var(--transition-base)',
          }}
          onMouseEnter={(e) => {
            (e.currentTarget as HTMLAnchorElement).style.borderColor = 'var(--color-border-accent)'
          }}
          onMouseLeave={(e) => {
            (e.currentTarget as HTMLAnchorElement).style.borderColor = 'var(--color-border)'
          }}
        >
          Ver portfólio
        </m.a>
      </m.div>
    </LazyMotion>
  )
}
```
